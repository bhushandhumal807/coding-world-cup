Welcome to the CIB Coding Challenge 2015!

This year's challenge is to create algorithms (AIs) to control a team of players competing in a virtual football World Cup. AIs can be written in any language.

To enter please mail me: richard.s.shepherd@gmail.com. I will add you to a mailing list so that you can be notified of any API changes or bug fixes. If anything is unclear in this documentation or if you find any bugs in the framework , please get in touch so that I can update the documentation and fix the problems.

The closing date for entries is 2nd February 2015, so get coding!

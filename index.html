Welcome to the Christmas Coding Challenge 2014!

This year's challenge is to create algorithms (AIs) to control a team of players competing in a virtual football World Cup. AIs can be written in any language.

The closing date for entries is 2nd February 2015, so get coding!

Getting started

To enter please mail me: richard.s.shepherd@gmail.com. I will add you to a mailing list so that you can be notified of any API changes or bug fixes.

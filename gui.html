Running the GUI

There is a simple HTML GUI. To run it:

  • In app.js set: var playWithGUI = true;
  • Run the server as normal, ie launch it with node app.js.
  • The server will wait for the GUI to connect.
  • Launch coding-world-cup.html from the 'gui' folder.

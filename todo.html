Upcoming features

  • There will shortly be a GUI to let you watch the game being played. Until it is ready, you should look at the log output to see the score and the game progress.
  • A 'tournament' feature will be added to play multiple AIs against each other.
  • Penalties will be adding, in case of a draw.
  • A request will be added so that you can specify the team name and colors.
  • "Timewasting" detection, for example if a goalkeeper has the ball, but does not kick it.
